# 🛠️ Dev Utils

> 开发者日常工具箱 — 纯前端，无后台，数据不离本地。

[![Stars](https://img.shields.io/github/stars/youling-dev/dev-utils?style=social)](https://github.com/youling-dev/dev-utils)
[![License: MIT](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)

---

## ✨ 工具列表

| 工具 | 说明 |
|------|------|
| { } **JSON 格式化** | 格式化 / 压缩 / 验证 JSON 数据 |
| 🔗 **URL 编解码** | URL Encode / Decode 转换 |
| 📝 **正则测试** | 实时测试正则表达式匹配 |
| ⏰ **时间戳转换** | Unix 时间戳 ↔ 日期互相转换 |
| 🔐 **Base64 编解码** | Base64 编码 / 解码 |
| 🔑 **密码生成器** | 生成安全随机密码 |
| 🆔 **UUID 生成** | 生成 UUID v4 |
| 📊 **文本对比** | 对比两段文本的差异 |
| 🎨 **颜色转换** | HEX / RGB / HSL 互相转换 |

---

## 🚀 使用

1. 打开 `index.html` 在浏览器中运行
2. 或部署到 GitHub Pages / Vercel / Netlify

**零依赖，纯 HTML/CSS/JS，双击即用。**
