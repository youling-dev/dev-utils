# Contributing to DevUtils

Thanks for your interest in DevUtils! Welcome to submit Issues and Pull Requests.

## Reporting Issues

Please include:
- Browser and version
- Issue description and reproduction steps
- Screenshots (if applicable)

## Submitting Code

1. Fork this repo
2. Create a feature branch (`git checkout -b feature/amazing-feature`)
3. Commit your changes (`git commit -m 'Add amazing feature'`)
4. Push to the branch (`git push origin feature/amazing-feature`)
5. Open a Pull Request

## Development Tips

- Pure frontend project, modify HTML/CSS/JS directly
- Keep code clean, add comments
- Test before submitting

## Code of Conduct

Please be friendly and respectful to the community.
